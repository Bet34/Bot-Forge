# Bot Forge

**Build custom AI bots visually — no coding required.**

A clean, simple tool to mix Neural Cores, Personas, Knowledge, and Tools to create powerful, monetizable AI bots.

## How to Use

1. Click parts on the left to add them to your build
2. Watch the Forged Prompt update automatically on the right
3. Click **Test Your Bot** to chat with it
4. Click **Copy Prompt** to use it in ChatGPT, Claude, Grok, etc.

## Features
- Visual bot builder
- Automatic prompt generation
- Synergy scoring
- Instant testing
- Clean exportable prompts
